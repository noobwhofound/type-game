# type-game
a type game: faster you type, more points you get.




# GameModes


1. Normal : You are given a random word. Type it as fast as you can.You will get your accuracy and time after each word.



2. Rush : You are given a word. Type it as fast as you can. You will get your result after n amount of words which you specify as an user input.


NOTE that the timer starts when you start typing. So take a good look at the word before you start typing.
